## API Endpoints

Base URL: `https://your-collate-instance.com/api/v1`

### Common Operations
- `GET /databases` - List all databases
- `POST /tables` - Create a new table
- `GET /users/me` - Get current user info
- `PUT /glossaryTerms/{id}` - Update glossary term
