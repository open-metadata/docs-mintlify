# MySQL Connector

Connect Collate to your MySQL databases to extract metadata, lineage, and usage information.

## Prerequisites

- MySQL 5.7 or higher
- User with appropriate permissions
- Network connectivity from Collate to MySQL

## Required Permissions

Create a dedicated user for Collate:

```sql
CREATE USER 'collate_user'@'%' IDENTIFIED BY 'strong_password';

GRANT SELECT ON *.* TO 'collate_user'@'%';
GRANT SELECT ON INFORMATION_SCHEMA.* TO 'collate_user'@'%';
GRANT SELECT ON PERFORMANCE_SCHEMA.* TO 'collate_user'@'%';

-- For lineage extraction
GRANT SELECT ON mysql.slow_log TO 'collate_user'@'%';
GRANT SELECT ON mysql.general_log TO 'collate_user'@'%';
```

## Configuration

### Basic Configuration

```yaml
source:
  type: mysql
  serviceName: mysql-production
  serviceConnection:
    config:
      type: Mysql
      hostPort: mysql.company.com:3306
      username: collate_user
      password: strong_password
      database: employees
```

### SSL Configuration

```yaml
source:
  type: mysql
  serviceName: mysql-ssl
  serviceConnection:
    config:
      type: Mysql
      hostPort: mysql.company.com:3306
      username: collate_user
      password: strong_password
      database: employees
      connectionOptions:
        sslmode: require
        sslcert: /path/to/client-cert.pem
        sslkey: /path/to/client-key.pem
        sslrootcert: /path/to/ca-cert.pem
```

## Metadata Extraction

The MySQL connector extracts:

- **Databases and Schemas**
- **Tables and Views**
- **Columns with data types**
- **Primary and Foreign Keys**
- **Indexes and Constraints**
- **Table and Column Comments**

## Usage and Lineage

Enable query log analysis for usage statistics:

```sql
-- Enable general log (for usage stats)
SET GLOBAL general_log = 'ON';
SET GLOBAL general_log_file = '/var/log/mysql/general.log';

-- Enable slow query log (for performance analysis)
SET GLOBAL slow_query_log = 'ON';
SET GLOBAL slow_query_log_file = '/var/log/mysql/slow.log';
SET GLOBAL long_query_time = 2;
```

## Troubleshooting

<Accordion title="Connection timeout">
Check network connectivity and firewall rules:

```bash
telnet mysql.company.com 3306
```
</Accordion>

<Accordion title="Permission denied">
Verify user permissions:

```sql
SHOW GRANTS FOR 'collate_user'@'%';
```
</Accordion>

<Accordion title="SSL connection issues">
Verify SSL certificates and configuration:

```bash
mysql --ssl-mode=REQUIRED -h mysql.company.com -u collate_user -p
```
</Accordion>
