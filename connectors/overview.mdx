## Configuration

All connectors follow a similar configuration pattern:

```yaml
source:
  type: mysql
  serviceName: mysql-prod
  serviceConnection:
    config:
      type: Mysql
      hostPort: localhost:3306
      username: collate_user
      password: collate_password
      database: employees
```

## Metadata Extraction

Connectors extract various types of metadata:

- **Schema Information**: Tables, columns, data types
- **Relationships**: Foreign keys and table relationships  
- **Usage Statistics**: Query patterns and access frequency
- **Lineage**: Data flow between tables and views
- **Data Profiling**: Column statistics and data quality metrics
